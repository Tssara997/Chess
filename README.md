# Chess

Simple chess game written in C++.

## About

This project is a desktop chess application created in C++.  
The goal of the project was to practice object-oriented programming, game logic implementation, and software architecture.

The application includes core chess mechanics such as:

- legal piece movement,
- turn-based gameplay,
- board state management,
- move validation,
- capturing pieces,
- basic game rules.
